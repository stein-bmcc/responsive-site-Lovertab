CUNY Website, 2022, HTML/CSS/JavaScript

This project was created in Atom and uploaded to GitHub where it can be accessed on Netlify through, Loverta Brown: CUNY Website. The goal of this project was to create a site where students who are going into Animation, Graphic Design, or Web development are able to see which CUNY schools have majors related to these fields. This site is broken down into disciplines, degrees, and boroughs. These sections can be further broken down (i.e.: associate's vs bachelor's degrees). On this website, potential students are also able to apply to CUNY, and Financial Aid.
